Rust-Traverse

Rust-Traverse is a terminal based file manager. It includes all the basic features of a file manager. Other features include:

  • Fully keyboard driven.
  • Fuzzy finding for files.
  • Path navigation.
  • Configurable.
  • Tar, or zip, extractions.
  • Bookmarks.
  • File content previews.

Still in active development with 70+ Github stars.

Message

Message is a full-stack web application for real-time direct messaging. It includes user authentication, JWT implementation. Its main functionality is with websockets created with Go. It is built with React, Go, Gorilla, and PostgreSQL. Other features include:

  • Message persistence.
  • Responsive design.
  • Rich text editing.

App is live at Message.

Noted

Noted is a full-stack web application for taking notes. It includes CRUD operations for notes, folders, and users. It also includes a search feature for notes. It is built with React, Node.js, Express, and MongoDB. Other features include:

  • Authentication.
  • Responsive design.
  • Rich text editing.

App is live at Noted.
